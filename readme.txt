PagTur Payment Plugin for WooCommerce

== Description ==
Accept Brazilian credit cards with installments up to 12x.
Automatic exchange USD x BRL and EUR x BRL.

== Installation ==
To install PagTur, go to the Plugins menu and select "Add New", look for "PagTur for WooCommerce" and click "Install Now".
After installation, you just need to activate and fill your integration data to can retrieve your Exchange Rates and sell even more.
